Pixels
It is the smallest controllable element of a picture represented on the screen.
Resolution
Screen resolution refers to the number of pixels a screen can display within a given area.
Lossy Compression
A way to make a file smaller that loses data that is compressed.
Lossless Compression
A way to make a file smaller that does not lose data that is compressed.
JPEG
A JPEG is an acronym for Joint Photographic Expert Groups and it is a 24 bit image format. It can hold 16.8 milion colors. JPEG uses lossy compression so when compressed you lose file data to make it smaller.
PNG
A PNG is an acronym for Portable Network Graphics and it has both an 8 bit and a 24 bit image format named PNG-8 and PNG-24. It supports lossless compressiong so that data isnt lost when compressed however this leads to a much larger file size so it should only be used when qua;ity of an image is what matters. It has better transperancy than GIF and supports image brightness, color correction, and 2 dimentional interlacing. PNG's cannot be animated and not all web site support them.
SVG
A SVG is an acronym for Scalable Vector Graphics and are very scalable. They are written in XML-Based Markup which is like HTML. They can be modified using CSS and JavaScript, and can be animated. For non-complex images SVG isnt the best choice because the size of the file is based of how complex the image is not how large it is.
GIF
A GIF is an acronym for Graphics Interchange Format. It only supports 256 colors so it isnt the best options for images with a lot of color density. It supports lossless compression so data isnt lost when compressing the file. For many years it was the only image format that supported transparency.
